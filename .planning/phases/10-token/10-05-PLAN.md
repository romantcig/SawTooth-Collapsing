<threat_model>
## Trust Boundaries

| Boundary | Description |
|---|---|
| pressureDecision → debugFact | 内存中的指纹与请求上下文只能映射为允许的数字/bool/enum |
| response usage → audit join | actual 必须是 deflation 前完整值，失败不能伪造账单 |
| request-scoped facts → filesystem | 并发 stage 文件不能覆盖或串 request_id |
| request context → terminal log | system/tools/messages/title/header/base64/凭证和敏感身份不得泄漏 |
| auxiliary response → observability | 可保留 usage 但不能声称更新主 baseline 或输出主压力摘要 |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|---|---|---|---|---|---|
| T-10-05-01 | Information Disclosure | debugFact / slog | high | mitigate | 扁平显式白名单、完整 sentinel 扫描、禁止 fingerprint/body/header/session 字段 |
| T-10-05-02 | Repudiation | request_id stage join | high | mitigate | pressure/usage 独立 sync.Once、原子文件名含 request ID/stage、join 回归 |
| T-10-05-03 | Tampering | actual_minus_selected | medium | mitigate | 只由 deflation 前 total 与已锁定 selected pressure计算，饱和有符号差值测试 |
| T-10-05-04 | Spoofing | auxiliary baseline outcome | high | mitigate | title/subagent response_usage 强制 baseline_updated=false 且无 pressure_decision/summary |
| T-10-05-05 | Denial of Service | debug/log volume | medium | mitigate | 每 stage sync.Once、每 main 请求最多一条摘要、无逐 block Info 日志 |
| T-10-05-06 | Information Disclosure | debug filesystem | high | mitigate | 继续使用 safe session hash 目录和原子 0755/文件写入路径；facts 本身不含 session ID |
</threat_model>
