## Trust Boundaries

| Boundary | Description |
|---|---|
| 上游 usage → SawtoothTrigger | 不可信 JSON/SSE usage 只有经现有非负、饱和汇总后才能成为 actual |
| 请求 system/tools → 持久 baseline | 敏感正文只能转成固定长度指纹，不能进入 SQLite、日志或 Debug |
| SQLite proxy_state → 内存 baseline | 冷启动数据可能旧版、畸形或部分缺失，必须安全降级 |
| 并发请求 → per-session baseline | actual、坐标和指纹必须以原子快照避免错误触发 |
