<threat_model>
## Trust Boundaries

| Boundary | Description |
|---|---|
| request messages/system/tools → pressureDecision | 不可信复杂 JSON 必须只产生计数与指纹，不泄漏正文或产生不稳定 fingerprint |
| persisted actual baseline → current decision | 旧 actual 只能在坐标增长且指纹一致时参与 delta |
| auxiliary classification → stateful pipeline | title/subagent 必须在所有主状态读取/写入前旁路 |
| upstream JSON/SSE usage → next baseline | 仅合法成功 usage、deflation 前 total 可更新主状态 |
| pressureDecision → Collapse pipeline | 进入决策与实际 Collapse 必须分离，避免错误强制折叠 |
