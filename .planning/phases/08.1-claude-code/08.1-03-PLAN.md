## ASVS Policy

- OWASP ASVS Level 1；阻断阈值为 high。
- 本轮两个 high 风险分别位于日志敏感信息边界与畸形辅助输入分类边界，均必须 mitigate 后才能完成。

## Trust Boundaries

| Boundary | Description |
|---|---|
| Claude Code headers/request body → requestMeta/logging | 原始 session ID、正文、system 与凭证属于不可信敏感输入，只允许进入通用受控日志上下文，不能进入字段受限辅助审计 |
| User content → auxiliary classifier | 畸形或多顶层 session 标签不能伪装成高置信自动标题请求并绕过 Sawtooth 状态管线 |
| Auxiliary classification → HandleMessages early return | 只有满足完整平衡 envelope 与既有 system/schema 门禁的请求才可获得无状态原字节旁路 |
